# OS Final Project

This repository contains our group materials for:
- Program 3 solution for the Tecnológico project

## Folders
- `P3/` = shared Program 3 solution

## Description
This program counts the number of valid triangles formed from a set of 2D points.
## Compilation
javac com/tryright/*.java
## Running the Program
java com.tryright.ThreadTriangles input.txt 4
## Input Format
First line: number of points
Following lines: x y coordinates
## Example
Input:
3
0 0
1 0
0 1
Output:
1
## Notes
Supports multi-threaded and multi-process versions
Tested with provided sample files
